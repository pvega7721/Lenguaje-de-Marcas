La World Wide Web (del inglés, Telaraña Mundial), la Web WWW.
es un sistema de hipertexto que funciona sobre Internet.
Para ver la información se utiliza una aplicación llamada navegador
web para extraer elementos de información (llamados "documentos" o "páginas web")
de los servidores web (o "sitios") y mostrarlos en la pantalla del usuario.
El usuario puede entonces seguir hiperenlaces que hay en la página
a otros documentos o incluso enviar información al servidor para interactuar
con él.

Y en esta asignatura nos moveremos por la WWW.
